About Canny Edge Detection

The Canny algorithm, invented by John F. Canny in 1986, is a powerful edge detection method. It works by:

  1. Reducing noise
  2. Finding intensity gradients
  3. Applying non-maximum suppression
  4. Using double threshold
  5. Tracking edges by hysteresis

This creates clear, thin edges perfect for stencils while preserving image quality.
